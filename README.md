# VisioFace
App that detects faces and body poses and displays them on the screen in real time

# System requirements
- iOS 15.0+

# Tech stack
- Language: Swift 6
- ML Framework: Apple Vision framework
- UI Frameworks: UIKit, AVFoundation, CoreAnimation
